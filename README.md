# 📊 Machine Learning Evaluation Metrics

## 📌 Objective

This repository explains and demonstrates evaluation metrics used in:

- Regression Models
- Classification Models

Understanding metrics is essential for selecting the right model 
and interpreting its performance correctly.

---

## 📈 Regression Metrics

Dataset Used:
California Housing Dataset

Metrics Covered:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## 📊 Classification Metrics

Dataset Used:
Breast Cancer Wisconsin Dataset

Metrics Covered:
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
- ROC Curve
- AUC

---

## 🔄 Key Takeaways

- MAE vs MSE differ in penalty for large errors.
- R² measures explained variance.
- Accuracy can be misleading for imbalanced datasets.
- Precision and Recall have a tradeoff.
- ROC-AUC measures classification performance across thresholds.
